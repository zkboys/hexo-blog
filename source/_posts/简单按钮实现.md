按钮一般分为三种状态，普通 鼠标移入 点击，一般状态切换，只是改变背景颜色，字体颜色，鼠标移入通过`:hover`伪类实现，点击使用`:active`伪类实现，简单的代码如下：
```css
.btn {
    display: inline-block;
    padding: 8px 10px;
    background: blue;
    border-radius: 5px;
    color: #fff;
    border: 0;
    box-sizing: border-box;
    outline: 0;
    cursor: pointer;
    text-decoration: none;
    transition: .3s;
}
/* hover 要在active 前面*/

.btn:hover {
    background: red;
}
.btn:active {
    background: yellow;
    color: #000;
}
```
